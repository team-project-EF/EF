# 오디션 자동화 AI 솔루션

### 팀 : EF (Entertainment Foundators)

---
##### 문제정의 : 
```
한해 오디션 지원자 최대 14만명으로 심사인력이 부족할 것이다.
```

##### 가설 : 
```
14만명에 달하는 쏟아지는 K-pop 오디션 지원자들을 인력에 제한받지 않고
1차 심사를 AI로 진행한다면 빠른 선정이 가능할 것이다.
```
